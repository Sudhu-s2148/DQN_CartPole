# DQN_CartPole
This project is a continuation of the RL agent using Qtable to solve the maze. Here we will be using a DQN to aid the agent on balancing a cartpole.
Currently this repo has all the necessry files to train and execute the process. 
in order to run this, 
  pull train, agent and buffer. 
  install the necessary modules.
  in the beginning of train file there will be a variable called training. set it to True if u want to train the network. Set it to false if u want to execute the trained network
  there is a variable called save_path. enter your desired name for the path in which the weights of the trained network will be stored with the specific reward system. when you run the train.py with training as False it will access the file in which the trained network was stored and run that.
